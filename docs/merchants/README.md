# Merchant documentation

**Audience:** merchant · **Arabic:** [README.ar.md](README.ar.md)

Step-by-step help for store owners lives **inside the running product** so it stays next to the UI: open **`/en/docs`** or **`/ar/docs`** on your Souqna deployment (routes under `src/app/[locale]/docs/`).

Merchant help should visually match the homepage: cream and charcoal surfaces,
thin grid dividers, Exo 2 English copy, Thmanyah Arabic headings, monochrome
integration marks, and no orange/purple/blue SaaS accents.

This repository’s [`docs/founder/`](../founder/) folder is aimed at founders and engineers maintaining the platform, not at day-to-day merchant tasks.
