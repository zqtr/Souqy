# توثيق التجّار

**Audience:** merchant · **English:** [README.md](README.md)

الإرشادات التفصيلية لأصحاب المتاجر تُعرَض **داخل المنتج** بجانب الواجهة: افتح **`/ar/docs`** أو **`/en/docs`** على نطاق Souqna (المسارات تحت `src/app/[locale]/docs/`).

مساعدة التجّار يجب أن تطابق الصفحة الرئيسية بصرياً: أسطح كريمية وفحمية،
فواصل شبكة رفيعة، Exo 2 للإنجليزية، Thmanyah لعناوين العربية، شعارات
تكاملات أحادية اللون، ومن دون ألوان برتقالية أو بنفسجية أو زرقاء بنمط SaaS
عام.

مجلد [`docs/founder/`](../founder/) في المستودع موجّه للمؤسّسين والمهندسين، وليس لمهام التاجر اليومية.
